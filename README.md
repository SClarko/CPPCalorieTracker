# CPPCalorieTracker
C++ Calorie Tracking system writtem in C++

Features:
- Barcode based food lookup
- Nutritional database integration
- Daily intake tracking
- Report generation

Tech:
- C++
- STL
- File I/O 
- Git version control

## Build (Windows / MSVC)

## Quick Start (Windows)

Option 1 (Easiest):
- Double-click `run.bat`

Option 2 (PowerShell):
- .\build.ps1
- .\run.ps1

Option 3 (Manually):
- mkdir build
- cd build
- cmake ..
- cmake --build . --config Debug
